How to DDoS?

Using the req command, format it as follows:
req <URL> <duration>.
The <URL> is the full URL of the target you want to send requests to, including the protocol (either http:// or https://), e.g., http://127.0.0.1:5000.
The <duration> specifies the duration (in seconds) for which the requests will be sent to the target URL.
For example, to send requests for 60 seconds, specify 60.
Example usage:
req http://127.0.0.1:5000 60 will send requests to http://127.0.0.1:5000 for 60 seconds.
